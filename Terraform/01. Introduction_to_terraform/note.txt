Installing Terraform
1. Download binary (Zipfile) form here: https://www.terraform.io/downloads.html
2. unzip the file downloaded and rename to terraform, 
3. cut the file and paste in your program file
4. Add the installation folder in your PATH (C:\Program Files\terraform)
5. confirm installation on cmd with terraform --version (Terraform v1.14.6)

Key Terraform Commands
1. Init -- Get me the providers I need
2. Plan-- What am I about to do
3. Apply -- Do what is in the tf files
4. Destroy -- Remover everything defined in the tf files
5. fmt -- formart your entire terraform scripts
